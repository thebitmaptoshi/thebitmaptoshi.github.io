Overview

Open Bitmap Internet is a Chrome browser extension that enables users to access Bitcoin inscription content through .bitmap domain names. This privacy policy explains how our extension handles data and protects user privacy.

What Our Extension Does

Our extension operates entirely locally in your browser and only:

  • ‚úÖFetches public Bitcoin inscription content from ordinals.com APIs when you visit .bitmap domains
  • ‚úÖTemporarily caches inscription content in browser memory for display purposes
  • ‚úÖUses session storage only to prevent infinite redirect loops (automatically cleared when you close your browser)
  • ‚úÖProcesses public blockchain data that is already available to everyone on the Bitcoin network

Data Sources

The extension only accesses:

  • Public Bitcoin inscription data from ordinals.com
  • Public registry data from GitHub repositories (thebitmaptoshi/BNS)
  • No private or personal user data of any kind

Third-Party Services

Our extension communicates with these public services:

  • ordinals.com - To fetch Bitcoin inscription content
  • ordiscan.com - To fetch Bitcoin inscription content
  • GitHub (raw.githubusercontent.com) - To access the .bitmap registry

These services may have their own privacy policies. We do not share any user data with these services.

Browser Permissions

Our extension requires certain Chrome permissions to function:

  • activeTab: To enhance Bitcoin inscription viewing on ordinals.com pages
  • scripting: To safely process and display inscription content
  • declarativeNetRequest: To redirect .bitmap domains to Bitcoin inscriptions
  • webNavigation: To intercept .bitmap domain requests
  • storage: For temporary session data only (redirect loop prevention)
  • host permissions: To fetch public inscription content and modify security headers

These permissions are used solely for the extension's core functionality and never for data collection.

Data Security

  • All processing occurs locally in your browser
  • No data is transmitted to external servers except for fetching public Bitcoin inscription content
  • No persistent storage of user information
  • Temporary session data is automatically cleared when you close your browser

Children's Privacy

This extension does not collect any personal information and is safe for users of all ages. We do not knowingly collect personal information from children under 13.
